# Tasks

Поточні та заплановані задачі.

## Статуси

- `[ ]` — не розпочато
- `[~]` — в процесі
- `[x]` — виконано

## Файли

- `README.md` — правила ведення задач у цьому розділі
- `sprint-*.md` — задачі по спринтах (за потреби)
- `backlog.md` — можна додати пізніше, якщо проєкту потрібен окремий загальний беклог
